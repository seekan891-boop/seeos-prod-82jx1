# SeeOs - YKS Çalışma Asistanı

## Kurulum (2 dakika)

### 1. Python yükle (yoksa)
- https://www.python.org/downloads/
- "Add Python to PATH" kutucuğunu işaretle

### 2. Terminal aç ve şunu yaz:
```
cd SeeOs
pip install -r requirements.txt
python app.py
```

### 3. Tarayıcıda aç:
```
http://localhost:5050
```

## DeepSeek AI (opsiyonel)
- https://platform.deepseek.com/api_keys adresinden API anahtarı al
- AI Koç sekmesine gir

## Dosyalar
- `app.py` - Ana uygulama
- `templates/` - HTML sayfalar
- `static/` - CSS, JS, ikonlar
- `.env` - API anahtarın (gizli, paylaşma!)
- `*.json` - Verilerin otomatik oluşur

## Durdurmak için
Terminalde `Ctrl+C`
